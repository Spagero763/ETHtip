# **App Name**: EthTip

## Core Features:

- Sub-Account Creation: Automatically create a sub-account for each user upon connecting their wallet using the Base Account SDK. This enables frictionless transactions, with no repeated signing prompts.
- Tip Submission: Allow users to enter an Ethereum address and an amount to tip (defaulting to 0.000001 ETH).
- Transaction Processing: Process tip transactions using the sub-account, leveraging spend permissions for gasless transactions from the user's Base Account balance.
- Transaction Confirmation: Display confirmation of successful tip transactions with transaction IDs. Consider tool for failed transactions and offering appropriate guidance
- Spend Permission Management: Enable users to manage and revoke spend permissions granted to the application via a dedicated settings panel.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), inspired by the concept of trustless, secure exchange.
- Background color: Light gray (#F0F2F5), providing a clean and neutral backdrop.
- Accent color: Electric purple (#BE29E2), for highlighting key actions and information, drawing on themes of cutting edge tech.
- Font: 'Inter', a sans-serif font known for its clear and modern design, used for both body and headlines.
- Use a set of simple, geometric icons to represent transaction types and settings.
- Maintain a clean and intuitive layout, with clear calls to action and easily accessible settings.
- Use subtle animations and transitions to provide visual feedback and enhance user experience.
